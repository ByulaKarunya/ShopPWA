# 🛍️ ShopPWA - E-Commerce Progressive Web App

ShopPWA is a modern, responsive, and installable **Progressive Web App (PWA)** for a simple e-commerce shopping experience. It supports offline browsing, push notifications, add-to-cart functionality, and product filtering – all built using plain JavaScript, HTML, Tailwind CSS, and localStorage.

## 🔥 Features

- 🛒 Add-to-Cart with quantity management
- 🔎 Product search (desktop & mobile)
- 📦 Offline support with Service Workers
- 📲 Installable PWA with custom install prompt
- 🔔 Push notifications using Web API
- 🌐 Connection status detection (online/offline)
- 🎨 Responsive UI with Tailwind CSS
- 💾 LocalStorage for cart & product caching
